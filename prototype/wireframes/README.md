# Wireframes Directory

This directory should contain all wireframes and mockups created during the hackathon.

## What to Add

### File Types Welcome:
- Figma exports (.png, .jpg, .pdf)
- Sketch files or exports
- Adobe XD exports
- Hand-drawn sketches (photographed/scanned)
- Balsamiq mockups
- Any other design artifacts

### Suggested Organization:

```

│   ├── splash-screen.png
```

## Instructions for Contributors

1. **Name files descriptively** – Use clear names like `parent-dashboard-home.png` instead of `screen1.png`
2. **Include annotations** – If possible, add notes about interactions or functionality
3. **Maintain consistency** – Use similar file formats and naming conventions
4. **Add context** – Create a summary document explaining the user journey
